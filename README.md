### Удаляет «мусорные» пробелы в тексте/коде

Сжимает подряд идущие пробелы между символами до одного и убирает хвостовые пробелы на концах строк.

#### Пример

**До:**
```cpp
std::cout      <<    "value"    <<    x    <<    std::endl;    
```

**После:**
```cpp
std::cout << "value" << x << std::endl;
```
